# AI in Robotics (Path Planning)

## Learning Objectives

- Robotics mein Artificial Intelligence (AI) ke kirdar ko samajhna.
- Path planning algorithms aur unke istemalat ka mutala karna.

## Core Concepts

AI robotics ka ek ahem juzv hai, khaas tor par path planning mein. Path planning algorithms robot ko ek maqam se doosre maqam tak mahfuz aur asar angaiz tareeqe se jane ka rasta talash karne mein madad dete hain, rukawaton se bach kar. A* (A-star) aur RRT (Rapidly-exploring Random Tree) jaise algorithms aam istemal hote hain.

## Chapter Summary

AI, khaas tor par path planning mein, robots ko khud mukhtar aur zaheen banane mein bunyadi kirdar ada karta hai. Advanced path planning algorithms robots ko complex maholon mein behtar tareeqe se navigate karne ki ijazat dete hain.

## Glossary Terms

- **Path Planning**: Robot ke liye ek maqam se doosre maqam tak mahfuz aur asar angaiz rasta talash karne ka amal.
- **A* Algorithm**: Ek heuristic search algorithm jo shortest path talash karne ke liye istemal hota hai.
- **RRT (Rapidly-exploring Random Tree)**: Ek algorithm jo complex maholon mein path planning ke liye istemal hota hai.
