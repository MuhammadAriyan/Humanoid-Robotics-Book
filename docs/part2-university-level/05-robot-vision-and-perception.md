# Robot Vision and Perception

## Learning Objectives

- Robot vision systems ke bunyadi usoolon ko samajhna.
- Image processing aur object recognition techniques ka mutala karna.

## Core Concepts

Robot vision robots ko apne mahol ko dekhne aur samajhne ki salahiyat deta hai. Is mein cameras se tasveerain hasil karna, unko process karna, aur phir un tasweeron se ahem maloomat (jese objects ki pehchan, unka fasla, aur unki shakal) nikalna shamil hai. Object recognition, segmentation, aur 3D reconstruction ahem techniques hain.

## Chapter Summary

Robot vision aur perception robots ko mahol ko dekhne, samajhne, aur uske mutabiq harkat karne mein madad dete hain. Advanced image processing aur object recognition techniques ke zariye robots mazeed zaheen aur khud mukhtar ban sakte hain.

## Glossary Terms

- **Image Processing**: Digital images ko behtar banane ya unse maloomat nikalne ke liye mukhtalif algorithms ka istemal.
- **Object Recognition**: Tasweeron ya videos mein khaas cheezon ko pehchanne ki salahiyat.
- **Segmentation**: Tasveer ko mukhtalif hisson mein taqseem karna taake har hissa kisi khaas object ya mahol ke juzv ko zahir kare.
