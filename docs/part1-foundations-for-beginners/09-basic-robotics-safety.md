# Basic Robotics Safety

## Learning Objectives

- Robotics ke bunyadi hifazati usoolon ko samajhna.
- Robots ke saath kaam karte waqt aam khatrat ko pehchanna.

## Core Concepts

Robots ke saath kaam karte waqt hifazat intehai zaroori hai. Is mein hifazati gawahi, emergency stop procedures, aur robot ke working space ko samajhna shamil hai. Hamesha zaroori hifazati eqdamaat ikhtiyar karne chahiye taake hifazat ya mali nuqsan se bacha ja sake.

## Chapter Summary

Basic robotics safety principles robots ke saath kaam karne walon ke liye intehai ahem hain. Hifazati eqdamaat aur khatrat ki pehchan ke zariye, aik mahfuz aur asar angaiz mahol banaya ja sakta hai.

## Glossary Terms

- **Emergency Stop (E-Stop)**: Ek hifazati button ya system jo robot ko fori tor par rok deta hai kisi khatre ki surat mein.
- **Working Envelope**: Robot ka wo area jismein wo harkat kar sakta hai, aur jismein insaanon ko dakhil hone se bachna chahiye.
