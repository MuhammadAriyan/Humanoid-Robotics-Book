# Building a Simple Robot (Conceptual)

## Learning Objectives

- Robot banane ke bunyadi marhalon ko samajhna.
- Ek simple robot ke design aur function ka tasawwur hasil karna.

## Core Concepts

Ek simple robot banana mukhtalif marhalon par mushtamil hota hai, jismein design, components ka intikhab, wiring, aur programming shamil hain. Is chapter mein hum ek bunyadi mobile robot banane ka tasawwurati khaka pesh karenge, jismein motors, wheels, sensors, aur ek microcontroller shamil honge.

## Glossary Terms

- **Microcontroller**: Ek choti computer chip jo embedded systems mein istemal hoti hai, jese Arduino ya Raspberry Pi.
- **Chassis**: Robot ka dhancha jo uske ajzaa ko sahara deta hai.
