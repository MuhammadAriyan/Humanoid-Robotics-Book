# Introduction to Control Systems

## Learning Objectives

- Control systems ke bunyadi tasawwurat ko samajhna.
- Open-loop aur closed-loop control ke farq ko pehchanna.

## Core Concepts

Control systems robots ko apne maqasid tak pohnchne mein madad dete hain. Open-loop control mein robot sirf pehle se program kiye gaye hukumat par amal karta hai, jabke closed-loop control mein sensors ke zariye feedback hasil kiya jata hai taake harkat ko durust kiya ja sake. Closed-loop systems ziada sahi aur adaptive hote hain.

## Chapter Summary

Control systems robots ko efficiently kaam karne mein madad dete hain. Open-loop aur closed-loop control ke zariye robots ko mukhtalif tareeqon se control kiya ja sakta hai, jismein closed-loop systems feedback ka istemal karte hain.

## Glossary Terms

- **Open-loop Control**: Ek control system jismein output par feedback nahi liya jata aur harkat pehle se tai shuda hoti hai.
- **Closed-loop Control**: Ek control system jismein sensors se feedback liya jata hai taake output ko maqsad ke mutabiq durust kiya ja sake.
